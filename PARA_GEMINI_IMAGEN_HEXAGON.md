## 🎯 PROMPT PARA GEMINI (COPIAR Y PEGAR)

```
Necesito que generes un DIAGRAMA HEXAGONAL profesional para la arquitectura de un sistema de monitoreo ambiental llamado GreenTech Amazon Systems.

REQUISITOS DEL DIAGRAMA:
- Forma: Hexágono regular
- Centro: "GreenTech\nArquitectura Hexagonal"
- 6 componentes en los 6 vértices:

1. PRESENTATION LAYER (Arriba)
   - Frontend: HTML5, CSS3, JavaScript
   - Archivos: index.html, app-sqlite.js
   - UI: Dashboard, Gráficos, Mapas

2. APPLICATION LAYER (Arriba-Derecha)
   - Lógica de Negocio
   - Gestión de Estado
   - Casos de Uso
   - Manejo de Eventos

3. API GATEWAY (Abajo-Derecha)
   - Express.js Server
   - REST API Endpoints
   - Autenticación Middleware
   - CORS, Validación

4. DATABASE LAYER (Abajo)
   - SQLite3
   - Tablas: users, sessions, kpis, alerts, chart_data
   - Persistencia Completa

5. INFRASTRUCTURE LAYER (Abajo-Izquierda)
   - Configuración Servidor
   - Utilidades y Helpers
   - Gestión Recursos
   - Dependencias npm

6. DOMAIN LAYER (Arriba-Izquierda)
   - Reglas de Negocio
   - Entidades Dominio
   - Validaciones
   - Estados y Constantes

CARACTERÍSTICAS DEL DIAGRAMA:
- Flechas bidireccionales entre componentes adyacentes
- Cada componente tiene color diferente (azul, púrpura, verde, rojo, naranja, amarillo)
- Centro con fondo gris claro
- Bordes bien definidos
- Iconos o símbolos representativos en cada componente
- Fuente clara y legible
- Estilo moderno, profesional y limpio
- Tamaño de imagen: 1200x1000 píxeles mínimo

FLUJO DE COMUNICACIÓN:
Presentation ↔ Application ↔ API Gateway ↔ Database ↔ Infrastructure ↔ Domain ↔ Presentation

FORMATO:
Genera una imagen PNG de alta calidad que pueda ser usada en presentaciones y documentación.

POR FAVOR, ASEGÚRATE DE:
✓ Que sea un HEXÁGONO REGULAR (no distorsionado)
✓ Que los 6 componentes estén claramente nombrados
✓ Que haya flechas conectando cada capa
✓ Que el centro sea visible y legible
✓ Que los colores sean vibrantes pero profesionales
✓ Que sea escalable (suficientemente grande y nítida)
```

## 📱 CÓMO USAR EN GEMINI

### Paso 1: Abre Gemini
```
https://gemini.google.com
```

### Paso 2: Nuevo Chat
- Click en "New Chat" o crea conversación nueva
- Asegúrate de tener habilitada la generación de imágenes

### Paso 3: Copia el Prompt
- Copia TODO el texto de la sección "PROMPT PARA GEMINI" (incluido)
- Ctrl+C

### Paso 4: Pega en Gemini
- Click en el campo de texto de Gemini
- Ctrl+V (pega)

### Paso 5: Envía
- Presiona Enter o click en el botón de enviar

### Paso 6: Espera
- Gemini procesará el prompt
- Generará el diagrama hexagonal
- Puede tomar 30-60 segundos

### Paso 7: Descarga
- Click derecho en la imagen generada
- "Descargar imagen" o "Save image as..."
- Guarda en: `c:\GreeTech\hexagon-diagram.png`
